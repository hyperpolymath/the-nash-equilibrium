= The Nash Equilibrium — Roadmap
:toc:
:icons: font

== Current Status

*Phase: Design | Completion: 15% | Maturity: Experimental*

Co-developed by Jonathan D.A. Jewell and son. A 6X strategy board game where players
guide civilisations from barter economies to post-scarcity futures while navigating
planetary boundaries. The Nash Equilibrium mechanic — finding optimal strategies given
what others are doing — is the core game-theoretic tension.
