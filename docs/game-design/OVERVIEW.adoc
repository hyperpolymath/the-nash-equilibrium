:icons: font

== Executive Summary

Econosphere (working title: *The Nash Equilibrium*) is a 6X strategy board game
where players guide civilizations from barter economies to post-scarcity futures,
while navigating planetary boundaries that limit unsustainable growth. The game
balances educational depth with strategic gameplay, making it ideal for classrooms
and board game enthusiasts.

== Key Features

* *Era-based progression* — Agrarian -> Industrial -> Digital -> Post-Scarcity
* *Planetary boundaries mechanic* — climate change, biosphere integrity, etc.
* *Multiple victory paths* — Capitalist, Socialist, Green, Techno-Utopian
* *Asymmetric roles* — unique constraints per player
* *Crisis events* — revolts, climate disasters, AI uprisings
* *6X pillars* — eXplore, eXpand, eXploit, eXterminate, eXchange, eXperiment

== Target Audience

* Economics students (secondary/university)
* Board game hobbyists (fans of Civilization, Terraforming Mars)
* Educators and futurists

== The 6X Connection

The game extends the classic 4X formula with two pillars that are central to the
Nash Equilibrium theme:

* *eXchange* — trade, diplomacy, game-theoretic negotiation where cooperation
  and betrayal coexist. Trade agreements and diplomatic pacts are modelled as
  repeated games with real mechanical consequences.
* *eXperiment* — speculative research with branching non-deterministic outcomes
  and asymmetric payoffs, creating information asymmetry that feeds into eXchange.

See link:6X-pillars.adoc[6X Pillars] for the full design rationale.
